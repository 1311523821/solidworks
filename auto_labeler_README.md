# auto_labeler.py — 智能特征打标与装配匹配

## 核心原则

**不猜零件类型，只看边的长度、类型、圆柱半径 —— 边长相等即匹配**

---

## 整体流程

```
./2/*.step
  → Step 1: 提取所有面的边特征（圆周长、线长）+ 圆柱面（半径、轴方向）
  → Step 2: 三层匹配（平面面 → 槽口面 → 圆柱面）
  → Step 3: 生成 *_label.json（坐标系标签）
  → verify_assembly.py: 链式对齐标签 → 装配 GLB
```

---

## Step 1：特征提取

遍历每个 STEP 零件的所有面，提取：

| 面类型 | 提取内容 |
|--------|---------|
| 平面 | 圆边（周长 = 2πr）、直线边（长度） |
| 圆柱面 | 半径、轴方向、轴中点、内外圆柱标记 |

---

## Step 2：三层匹配

### Step 2a: 平面面匹配（边长相等）

| 项目 | 说明 |
|------|------|
| 条件 | ≥3 特征直接过 / =2 特征须含圆（过滤线-线巧合） |
| 典型匹配 | 法兰面↔法兰面（9c+3l）、轴键槽↔键底（4l） |
| 原点 | 面的几何中心 |
| Z 轴 | A=面法向 / B=-(面法向)（面贴面时法向相反） |
| X 轴 | 面中心→第一个匹配圆/线的方向 |

### Step 2b: 槽口面匹配（孔轴过滤 + 宽松阈值）

| 项目 | 说明 |
|------|------|
| 筛选 | 面距孔轴 < r+15mm 且面法向⊥孔轴 |
| 条件 | ≥2 特征即可 |
| 典型匹配 | 轴槽面↔法兰槽面（2l，8mm 宽度边匹配） |
| **用途** | **只给 X 方向参考（旋转约束），不参与位置定位** |

### Step 2c: 圆柱面匹配（半径相等）

| 项目 | 说明 |
|------|------|
| 条件 | 外圆柱 R ≈ 内孔 R（公差 0.5mm） |
| 典型匹配 | 轴外径↔法兰内孔（R≈19.80） |
| 原点 | 轴=中点 / 孔=**轴线×法兰贴合面交点** |
| Z 轴 | 轴线方向 |
| **X 轴** | **轴心→槽口面中心（径向向量，无180°歧义）** |

---

## Step 3：标签格式

```json
{
  "modelAnnotation": {
    "features": {
      "featureCoordSyses": [{
        "identifier": "零件名_Mating_N",
        "geometry": {
          "origin": {"x": ..., "y": ..., "z": ...},
          "x": {"x": ..., "y": ..., "z": ...},
          "y": {"x": ..., "y": ..., "z": ...},
          "z": {"x": ..., "y": ..., "z": ...}
        },
        "userData": {"type": "MATE", "matchType": "PLANAR|CYLINDER", "total": N}
      }]
    }
  }
}
```

---

## 装配逻辑 (verify_assembly.py)

1. 锚点 = PLANAR 匹配品质最高的零件（面贴面最强约束）
2. 链式放置：已放置零件 → 未放置零件（标签对齐）
3. 品质评分：total≥10→score0, total≥3→score1, CYLINDER→score2, 弱匹配→score3（不参与定位）

---

## 匹配经验总结

| # | 经验 |
|---|------|
| 1 | **圆孔配圆孔，方孔配方孔**——边长相等即匹配，无需猜零件类型 |
| 2 | **三层匹配分层负责**：平面定位 → 槽口定旋转 → 圆柱定轴心 |
| 3 | **X 方向用轴心→面中心的径向向量**，避免180°歧义 |
| 4 | **孔面交点**作圆柱标签原点——两法兰孔面交点同在贴合面，自动重合 |
| 5 | **弱匹配（2特征）不给位置只给旋转**——面去重跳过，不影响链式定位 |
| 6 | **孔轴过滤筛选槽口面**：距孔轴 < r+15mm + 法向⊥孔轴，大幅减少噪声 |
| 7 | **所有阈值相对化**：`距轴面<r±8mm`、`孔壁<r±15mm`，不依赖具体尺寸 |
| 8 | **装配品质评分**：强匹配定位置(0,1) → 圆柱定轴心(2) → 弱匹配合弃(3) |
| 9 | **圆匹配 X 用面中心→匹配圆心**，线匹配 X 用径向向量（面法向/匹配线方向有180°歧义） |
| 10 | **两个法兰孔的圆柱标签原点自动重合**（都是轴线×贴合面交点），轴穿入即居中 |

## 参数说明

| 参数 | 值 | 含义 |
|------|-----|------|
| TOL | 0.1mm | 边长匹配公差 |
| MIN_OK | 3 | 平面匹配直接接受的特征数 |
| MIN_OK_LOOSE | 2 | 平面匹配最低特征数（需含圆） |
| SLOT_MIN | 2 | 槽口匹配最低特征数 |
| CYL_TOL | 0.5mm | 圆柱半径匹配公差（兼容间隙配合） |
